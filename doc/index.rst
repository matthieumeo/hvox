##################################################
HVOX: Mesh-agnostic wide-field interferometry
##################################################

*HVOX* is a Python 3 package for mesh-agnostic wide-field interferometry.
This software provides tools for radio astronomy data analysis and synthesis via the Heisenberg voxelisation method [HVOXpaper]_.
